# Sales Data Analysis using Excel

## Project Overview
This project analyzes retail sales data using Microsoft Excel to identify sales trends and performance across regions and categories.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Key Analysis Performed
- Region-wise sales analysis
- Category and sub-category sales comparison
- Identification of top performing regions and products
- Visual representation using charts

## Outcome
This project demonstrates practical Excel skills used in entry-level Data Analyst and MIS roles.
